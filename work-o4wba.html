Old 4th Ward Neighborhood Business Association Dashboard

TLDR: Used Tableau to create a dashboard for the creation of a new Commercial Improvement District attempt. Integrated Tableau into a git pages site linked it up with google sheets as the database for the dashboard.

Topics

Tableau Public, GeoJSON, Google Sheet API, CID, Tax Assessor Property Data

Project Motivation

This dashboard was the first time I had ever worked with property datasets, dashboards, and web infrastructure. The Old Fourth Ward Neighborhood Business Association came to propose a project for a student run consultant entity called Epic Intentions. They wanted to start a Community Improvement District in Old 4th Ward to help fund public projects like road improvement. However, they needed atleast 60% of business owners in the neighorhood to sign on to legally create the district. They had a need to visualize which owners had signed on and how that affected their total progress towards that 60% goal. With this task in mind and constraints of keeping the tool completely free since the association wanted to avoid costs in the long-term, we settled on using Tableau public(a free dashboard visualizer) integrated with github pages(a free site hosting solution) and google sheets API(a free database solution). The product ended up coming together and while it was rough around the edges it was a great learning experience for me in term of consulting, working with property data, and using web infrastructure.

 

 

The dashboard has the following features:
  • Input and Edit Business Owner Sign On for CID Approval
  • Toggle Map to see all properties in CID or those just soft commited/signed on
  • See demographics of who has signed on
  • See progress towards 60% goal of properties and percentage of assessed value in favor
  • Password protected permissions to edit the underlying data
