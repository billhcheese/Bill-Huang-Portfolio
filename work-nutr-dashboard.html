Food Nutrient Information Visualization Dashboard

TLDR: Created an interactive dashboard using Javascript D3 package to explore a food nutrients dataset

Topics

Interactive Visualization, Information Vis, D3, Javascript, HTML, CSS, Data Exploration, Web Development

Project Motivation

This dashboard was created as a final project where we were to choose a dataset and create an interactive visualtion of the dataset that would take advantage of D3's ability to create interactive interfaces. I wanted people to be able to explore using this visualization, and I enjoy food. Therefore, I created an interactive dashboard for users to explore food nutrients of different foods. It looks like the following picture.

 

 

The dashboard has the following features:
  • Scatterplot generated with colors accodring to food category
  • Tooltip displays on hover in scatterplot for food item details
  • Point map on the right will display nutrients of currently selected food item on scatterplot
  • Average value of nutrients is shown in point map for comparison of foods
  • Drop-down menu allows you to change the y-axis of scatter plot for exploring other nutrients
